Proof

Proof is a library of classes for creating object oriented applications in PHP. Objects are king here and are treated as such.

Why?

Because PHP is a scripting language.

Usage

To start using proof just copy the "proof" folder and "proof.php" file to your server route. From there just include the "proof.php" file and your on your way.
